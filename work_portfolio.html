Tech Stack Used


Technical Underpinnings:

This portfolio is the result of a harmonious blend of various technologies that bring it to life. The front-end is a fusion of HTML, CSS, and JavaScript, meticulously crafted to deliver an engaging user interface and a seamless user experience. JavaScript's power is harnessed to enhance interactivity and smooth animations. Additionally, I've incorporated jQuery to add extra dynamism to certain components.

To ensure a structured and organized approach, I've adopted a React-based structure. The portfolio is divided into distinct components, each with dedicated styling and scripting. This architectural choice allows for maintainability and scalability as the project evolves

1. Node.js & Express.js: Vite Dev Environment

On the server-side, I've employed Node.js and its powerful framework, Express.js to handle requests and responses efficiently, providing a robust and scalable foundation for this portfolio. Node.js enables high responsiveness and the ability to handle concurrent users with ease, ensuring a seamless experience for visitors.

In addition to the server-side setup, I have utilized Vite dev environment to optimize the front-end development process. Vite is a lightning-fast build tool and development server specifically designed for modern JavaScript projects. Thanks to Vite's ability to leverage native ES modules in modern browsers during development, the build process is significantly accelerated.

The lightweight and lightning-fast nature of this technology stack ensures a high-performing, responsive, and user-friendly web experience.


2. Dynamic Blog Section:

Enhancements and Limitations in Blog Post URL Structure:*
At present, the dynamic blogging section does not feature dedicated URLs for individual blog posts. This is due to the absence of a routing library or the current implementation, which is functioning seamlessly. While dedicated routing for the blog section is not in place now, I am considering the possibility of implementing it in the future to further enhance the user experience.

The blog section of this portfolio is empowered by STRAPI, an expressive and feature-rich open-source Content Management System (CMS). This dynamic platform allows me to seamlessly create and share captivating content, enhancing your reading experience. I'm excited to share compelling content and provide insights into my journey as a developer.


3. MySQL Database from Amazon Web Services - RDS:

To ensure data integrity and reliability, I've entrusted the data handling to a robust MySQL database hosted on Amazon Web Services - RDS. Leveraging the power of AWS ensures seamless data management, scalability, and high availability, allowing me to focus on delivering the best user experience.

My portfolio boasts a mobile-first, responsive single-column layout that seamlessly adapts to diverse screen sizes, whether on a desktop, tablet, or smartphone. Rooted in the core of my design philosophy, accessibility ensures that every user can effortlessly explore and enjoy my portfolio.
